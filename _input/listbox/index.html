Readme

Listbox is an ARIA and JavaScript alternative to HTML's select element; allowing custom HTML and custom styling. A listbox can also be opened from a button or combobox.

This example imports skin and makeup for base styles and behaviour respectively. Please read the documentation for further developer guidance.

Manual Selection

ARROW keys move the active descendant. SPACEBAR key is required to manually change the selected state.
