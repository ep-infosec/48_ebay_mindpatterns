Readme

A combobox allows the user to type their own value into the textbox or select a value for the texbox from a list of options.

This example imports skin and makeup for base styles and behaviour respectively. Please read the documentation for further developer guidance.

Please read the documentation to understand how these examples were created.

Examples

Manual Selection

With manual selection, ENTER key is required to manually update the textbox with the value of the current active descendant (controlled via arrow keys). This means that an additional ENTER key press is required to submit any associated form.

Automatic Selection

With automatic selection, ENTER key is not required, the textbox value is automatically updated to match the current active descendant option.
