Readme

Activate the button to open a listbox. Listbox-Button is an ARIA and JavaScript alternative to HTML's select element; allowing custom HTML and custom styling.

This example imports skin and makeup for base styles and behaviour respectively. Please read the documentation for further developer guidance.

NOTE: in these examples the button itself always contains a permanent accessible "label", but there are also other alternate ways to satisfy labelling requirements.

Manual Selection

ARROW KEYS move the active descendant. SPACEBAR key is required to manually change the selected state.
