Readme

A resultset can be paginated via server side page reloads (i.e. links) or client side page rendering (i.e. buttons).

Please read the documentation to understand how these examples were created.

Examples

Example 1: Server-Side Pagination

Use TAB keys to navigate links; ENTER key to activate.
