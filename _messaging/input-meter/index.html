Readme

Start typing in the textbox or textarea and the meter will update. Enable your screen reader to hear live updates at various intervals.

NOTE: the meter tag is not supported in Internet Explorer. A simple textual fallback would be required (but is not demonstrated here).

Please read the documentation to understand how these examples were created.
