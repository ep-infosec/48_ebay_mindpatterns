Readme

All notices should be discoverable via screen reader landmark navigation. Client side notices must be announced when rendered or made visible.

Please read the documentation to understand how these examples were created.

Examples

Example 1: Server Side Confirmation

This confirmation notice is discoverable via list of landmarks.

Nokia Lumia 635

Example 2: Server Side Information

This information notice is discoverable via list of landmarks.

Nokia Lumia 635

Example 3: Client Side Attention

Click "Add to Cart" button to display and announce notice.
